CMD注释

2016-06-22

  1. :: 注释内容（第一个冒号后也可以跟任何一个非字母数字的字符）

  2. rem 注释内容（不能出现重定向符号和管道符号）

  3. %注释内容%（可以用作行间注释，不能出现重定向符号和管道符号）

  4. :标签 注释内容（可以用作标签下方段的执行内容）
