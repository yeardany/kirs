Map 和 Object

  • Object 键只能是字符串或者符号,Map 可以是任意值，包括函数
  • Map 中键值是有序的，按照插入的顺序排列
  • 可以通过 Map size 属性获得键值个数，Object 则无法实现
  • Map 性能优于 Object
