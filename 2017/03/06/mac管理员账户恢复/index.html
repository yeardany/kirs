本来想改个用户名，结果直接输入个空格就保存了，重启后管理员账户没有了，变成了一个其他选项，瞬间懵逼了，各种试都进不去，看网上的帖子，有好多说要重装系统的，可我里面还存了很多资料岂不是说没就没了？琢磨一会儿后总算找到了解决办法：

单用户模式

进入单用户模式（Single User Model），清除启动配置，开机按住 cmd+s 直到出现命令行界面（字小的看的都要瞎了），输入一下命令

新建管理员

  • 执行上面的操作后，重启会发现和刚买来一样，这是因为新建了一个管理员 账户1 ，那原来我的配置去哪了？介于linux机制，其实每一个用户都会对应一个该用户名的文件夹，在终端运行 cd /Users 会罗列所有用户的文件夹，记住另外一个 账户2 文件夹名

  • 打开 系统偏好设置–>用户与群组->解锁->左下角加号 ，创建一个新账户设定为”管理员“全名任意，账户名（这将用作您个人文件夹的名称），注意这句话，这里填的就是刚才查看的 账户2 文件夹名，一定要一样，mac会询问已存在同名用户文件夹是否覆盖，选择是，这样就将新用户绑定到原有账户文件夹上，算是还原了。

  • 重启电脑，登录刚才新建的 账户2 名称的用户，是不是和原先一毛一样了？

  • 最后打开 系统偏好设置–>用户与群组->解锁->左下角减号 ,选择删除 账户1 就
