下载地址和安装方法可能随版本变化有所不同，仅供参考

Windows

下载

官网下载绿色版本的MySql，解压到除C盘之外的其他盘符，不然会报错。

配置

在 mysql-x.x.xx-winx64 目录下，新建个my.ini。my.ini会替换掉下面的my-default.ini文件。.ini文件是window里面的配置文件。保存里面各种默认的数据。安装版的是在安装的时候让你自己选然后系统给你保存进来，zip archive是自己写。这个文件在软件运行时加载（把里面D:\mysql\mysql-5.6.17-winx64改成自己的软件安装路径）
