Flex布局

Jul 13, 2021 | css | Hits

Flex布局是一种一维布局，其一次只能处理一个维度上的元素，即一行或者一列

flex布局属性

flex-direction轴线

Flex布局有两根轴线：主轴、交叉轴

主轴：由flex-direction定义，包含4个值：

  • row主轴横向
  • row-reverse主轴横向
  • column主轴纵向
  • column-reverse主轴横向

交叉轴：垂直于主轴

flex-wrap换行

  • nowrap：所有项目的子元素将会缩小以适应容器，涉及到flex-shrink，若项目的子元素无法缩小，使用nowrap会导致元素溢出容器
  • wrap：所有项目的子元素在容器宽度不够时换行显示

Flex容器

使用display:flex或者display:inline-flex创建Flex布局，默认状态下元素呈线形排列，并且把自己的大小作为主轴上的大小。如果有太多元素超出容器，它们会溢出而不会换行。如果一些元素比其他元素高，那么元素会沿交叉轴被拉伸来填满它的大小。

起始线和终止线

Flex布局并没有规定哪个方向是开始或终止，随着语言书写的不同，起始线和终止线的方向会相应变化，如英文，主轴的起始线是左边，终止线是右边。又如阿拉伯语，主轴的起始线是右边，终止线是左边。在这两种情况下，交叉轴的起始线是flex容器的顶部，终止线是底部，因为两种语言都是水平书写模式。

flex-flow简写

将flex-direction和flex-wrap组合为简写属性flex-flow，第一个指定的值为flex-direction，第二个指定的值为flex-wrap

flex元素属性

flex-basis

元素占用空间大小，默认值为auto，即随元素内容多少而变化宽度

flex-grow

正整数，以flex-basis为基础延展元素

flex-shrink

正整数，以flex-basis为基础收缩元素，只有在flex元素总和超出主轴才会生效

flex简写

把三个数值按这个顺序书写flex-grow，flex-shrink，flex-basis

预定的简写形式：

  • flex: initial等价于flex: 0 1 auto
  • flex: auto等价于flex: 1 1 auto
  • flex: none等价于flex: 0 0 auto
  • flex: 1等价于flex: 1 1 0

对齐方式

  • align-items设置元素在交叉轴上对齐方式
    • stretch默认值，拉伸填充flex容器，最高元素定义了容器的高度
    • flex-start顶部对齐
    • flex-end底部对齐
    • center中线对齐
  • justify-content设置元素主轴上对齐方式
    • flex-start默认值，从起始线开始排列，
    • flex-end从终止线开始排列
    • center从中间开始排列
    • space-between元素之间间隔相等
    • space-around元素左右间隔相等
