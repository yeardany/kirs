关于Mac工具

Jul 29, 2016 | essay | Hits

简单罗列下Mac下我安装的软件（想到那些到哪😄）

编程相关

  • Xcode：后续很多软件都依赖它，写写C、C++非常棒，后续可以写Swift
  • Sublime Text：支持非常多的格式，方便快速
  • WebStorm：Jetbrains全家桶之一，配合学校教育邮箱免费使用
  • MWeb：区分lite版和完整版，写Markdown很方便，还能直接推送到WordPress博客
  • iPic：配合MWeb将图片快速上传图床并得到地址
  • Paste：强大的剪贴板历史管理
  • Transmit：ftp/sftp客户端
  • Yummy FTP Alias：快速上传文件到服务器
  • Charles/Postman：网络封包/模拟请求工具
  • iTerm2：代替Mac自带的Terminal
  • Homebrew：OS X失散多年的包管理器😃。顺着homebrew
    `安装archey：brew install archey在终端显示电脑配置信息`
`安装nodejs：brew install node`
`安装jenv：brew install jenv java多版本管理`
`安装java：brew cask install java`
  • Dash：查询各种api文档

日常使用

  • Microsoft Remote Desktop：远程连接Windows
  • AirDroid：无需数据线，远程管理安卓手机
  • WhatsApp：微信过于臃肿，来个简单清爽的，界面也不错
  • Thunder：下载电影使用
  • Bartender 3：菜单栏整理工具
  • Flux：调节屏幕色温，保护眼睛
  • Pocket：稍后阅读，用于浏览网页收集资料
  • Encrypto：加密文件使用
  • InsomniaX：防止电脑闲置/盒盖休眠
  • UnRAR：压缩/解压
  • CCleaner：偶尔清理下Mac有钱的活可以用（CleanMyMac
  • Dr.Cleaner：已经使用Dr.Cleaner代替CCleaner
  • QQ音乐：虽然网易云音乐是种情怀，可是QQ貌似音源更多😓
  • WeiboX：很好用的微博客户端
  • 暴风影音：QuickTime无法播放的都能搞定，比windows版简单清爽很多
