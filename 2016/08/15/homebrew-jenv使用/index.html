即之前所写linux下java环境变量设置后，本文为涉及Mac下相关操作，分为两部分：Homebrew使用，java、jEnv安装使用，那么，开始吧

第一部分：Homebrew安装及使用

  1. 简介

    包管理工具，官方称之为The missing package manager for OS X。

  2. 安装

  1. 常用命令

    搜索包：brew search node

    查看包的主要信息：brew info node

    罗列所有已安装的包：brew list

    安装包：brew install node

    更新包源：brew update

    检查所装包是否有新版本：brew outdated

    升级所有软件：brew upgrade 或者 brew upgrade <包名>

    清理安装包缓存：brew cleanup

    另外，关于brew cask：brew-cask 允许你使用命令行安装 OS X 应用。比如你可以这样安装 Chrome：brew cask install google-chrome。还有 Evernote、Skype、Sublime Text、VirtualBox、java等都可以用 brew-cask 安装。

    brew install caskroom/cask/brew-cask

    当然，其实不用刻意去装caskroom，输入 brew cask install <软件名> homebrew会自动安装caskroom

  2. 当然，有时候项目需要非最新版本的java，那么就需要安装其他版本，cask也提供了相应的功能，

    brew tap caskroom/versions

    brew cask install java7

  3. 这样，就会安装上java8和java7两个版本，使用 brew cask list 查看，另外java安装路径为：/Library/Java/JavaVirtualMachines/
